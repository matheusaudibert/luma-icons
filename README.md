# luma-icons

Luma's icon set as React components. 879 icons, fully typed, tree-shakeable.

> [!NOTE]
> Not affiliated with or licensed by [Luma](https://lu.ma).

## Install

```bash
npm install luma-icons
```

## Usage

```tsx
import { Check16, Alarm24, BrandsDiscord } from 'luma-icons'

<Check16 />
<Alarm24 size={32} color="#6366f1" />
<BrandsDiscord size={20} />
```

Some icons (brand colors, payment, weather) preserve their original colors and don't accept the `color` prop.

Browse all icons at [Luma's official style guide](https://luma.com/style-guide/icons) or at https://grkg.github.io/luma-icons/.
Acces the npm's package at https://www.npmjs.com/package/luma-icons.
